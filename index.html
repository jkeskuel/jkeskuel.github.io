About

I'm Jure Kranjc, an independent database consultant specializing in SQL Server with extensive experience across corporate and enterprise environments. I deliver honest, professional solutions with a personal touch to every engagement.

My clients range from government institutions to private enterprises, where I architect and optimize highly available, efficient, and high-performance database systems that support critical business operations.

  • MCSA: SQL Server 2016

  • Microsoft Certified: Azure Data Engineer Associate

Services

Core database consulting services:

  • High Availability & Disaster Recovery: Design, implementation, and management of enterprise SQL Server HA/DR solutions
  • Database Migrations: Server consolidations, virtualization projects, and platform migrations with zero-downtime strategies
  • Performance Optimization: Query tuning, schema optimization, and application code improvements for maximum efficiency
  • Cost Optimization: Licensing analysis, resource right-sizing, and architectural recommendations to reduce total cost of ownership
  • Development Consulting: Expert guidance for data-intensive application development and database design best practices
  • DevOps & CI/CD: Database deployment automation, version control integration, and continuous delivery pipelines
  • Training & Mentoring: Knowledge transfer programs for individuals and teams to build internal database expertise
